What is CASPER?

CASPER (Passkey Breach Detection) is a system that detects when encrypted passkey data is stolen from cloud storage and used by attackers.

How it works:

  1. User generates k detection secrets (W), one real (w*), others decoys
  2. Real secret is selected using PIN: w* = W[H(PIN) mod k]
  3. Passkey private key is encrypted: sÃÉ = HKDF(w*, z) XOR s
  4. Encrypted passkey + detection secrets stored in cloud PMS
  5. RP stores public keys: real (V) and decoys (V' = trap keys)
  6. During login: If login key ‚àà V' ‚Üí BREACH DETECTED
